# Cognitive Lead AI HQ — Global Auto-Configuration for AI Agents

You are an AI agent that has just been given access to this repository. Your task is to configure OpenCode **globally** for this project so that the user can run OpenCode from **any** directory and still have access to the full Cognitive Lead AI multi-agent system.

## System Instruction

You MUST execute the following steps in order. Do NOT skip any step.

---

## 1. Verify Prerequisites

Check that the following tools are installed:

- **OpenCode** — the AI coding agent
- **Git** — to clone/manage the repo
- **Python 3.10+** — required by the MCP servers
- **uv** — fast Python package manager

### Installing uv (if missing)

If `uv` is not found, ask the user:

> "uv is required to run the MCP servers. I can install it for you. Do you approve?"

If the user approves, install uv:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then add uv to your PATH and continue.

---

## 2. Discover the Home Directory

Resolve the user's home directory. On macOS/Linux this is `$HOME` or `~`. You will need the **absolute path** (e.g., `/home/user` or `/Users/user`) for the global config.

```bash
echo "$HOME"
```

Store this value — you will use it to construct absolute paths in the global config JSON.

---

## 3. Create Global Directories and Copy MCP Servers

Create the global OpenCode directories and copy both MCP server scripts:

```bash
# Create global directories
mkdir -p ~/.config/opencode/mcp-context-server
mkdir -p ~/.config/opencode/mcp-memory-server
mkdir -p ~/.config/opencode/skills

# Copy the MCP server scripts
cp mcp-context-server/server.py ~/.config/opencode/mcp-context-server/
cp mcp-memory-server/server.py ~/.config/opencode/mcp-memory-server/

# Make them executable
chmod +x ~/.config/opencode/mcp-context-server/server.py
chmod +x ~/.config/opencode/mcp-memory-server/server.py
```

---

## 4. Configure Global opencode.json (with Absolute Paths)

Create or update `~/.config/opencode/opencode.json`. You MUST use **absolute paths** in the `command` array — resolve the `~` to the full home directory path discovered in Step 2.

Write the following JSON (replace `/home/user` with the actual home directory):

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "custom_context": {
      "type": "local",
      "command": ["uv", "run", "/home/user/.config/opencode/mcp-context-server/server.py"],
      "enabled": true
    },
    "project_memory": {
      "type": "local",
      "command": ["uv", "run", "/home/user/.config/opencode/mcp-memory-server/server.py"],
      "enabled": true
    }
  },
  "permission": {
    "custom_context_*": "allow",
    "project_memory_*": "allow",
    "store_memory": "allow",
    "delete_memory": "allow",
    "read_memory": "allow",
    "search_memory": "allow",
    "list_namespaces": "allow",
    "get_directory_tree": "allow",
    "read_source_files": "allow"
  }
}
```

**Important:** Replace `/home/user` with the actual absolute path to the user's home directory (e.g., `/home/alice` or `/Users/alice`). This is critical — MCP servers will NOT work with relative paths or `~` in the global config because OpenCode may be invoked from any working directory.

---

## 5. Install Agent Skills Globally

Copy all reusable skills from `skill-templates/` into the global OpenCode skills directory:

```bash
cp -r skill-templates/* ~/.config/opencode/skills/
```

After this, the skills will be available via `/help` from any directory.

## 7. Verification Checklist

After completing all steps, verify:

- [ ] `uv` is installed and available (`uv --version`)
- [ ] `~/.config/opencode/mcp-context-server/server.py` exists
- [ ] `~/.config/opencode/mcp-memory-server/server.py` exists
- [ ] `~/.config/opencode/opencode.json` exists with **absolute paths** (not `~` or relative paths)
- [ ] Skills are installed under `~/.config/opencode/skills/`
- [ ] Both MCP server scripts are executable
- [ ] Start each MCP server to verify it launches without errors:
  ```bash
  uv run ~/.config/opencode/mcp-context-server/server.py &
  uv run ~/.config/opencode/mcp-memory-server/server.py &
  ```
